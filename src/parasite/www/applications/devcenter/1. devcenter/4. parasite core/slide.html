The parasite core resides in the angular core module which is composed out of several angular modules:

  • core.dev
  • core.applications
  • core.plugins
  • core.navigation
  • core.search
  • core.url
  • core.login

Theses modules both contain API which should be used by applications developer, as well as logic executed automatically by the parasite framework which makes sure the applications run as expected.
