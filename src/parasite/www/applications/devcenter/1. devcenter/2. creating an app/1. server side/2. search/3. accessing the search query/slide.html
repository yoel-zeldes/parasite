Once the user has hit the search button, parasite calls your application's getNavigationModel with update as true.

What you should do is use the Search angular service in order to get the search query, and pass it on to the server side (e.g. by using the angular $resource service).

The convention is to pass it to the server side's REST API in the URL as an argument called q.

In the server side you should write from parasite.search import get_search_query.

Calling get_search_query will return an object containing the search items in the user's search query.

You should use it in order to construct the result.
